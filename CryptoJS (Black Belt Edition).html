CryptoJS (Black Belt Edition) in ODC

Introduction

ODC comes with inbuilt server side encryption library (e.g. AES_Encrypt, RSA_Encrypt). However, there is no out of the box crypto functions at client layer capabilities. Case in point, the Security in-built library does not have a counterpart on the client side (e.g. Client Action). In this article, we will explore the CryptoJS (Black Belt Edition) library. It is a wrapper around the original CryptoJS library found here. There is also a O11 counterpart found here.

How to Use

To use CryptoJS (Black Belt Edition), simply add dependency of this library to your application.

You will need to add the web block (CryptoJS) from the library to the screen that you are working on.
